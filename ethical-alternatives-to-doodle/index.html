Ethical alternatives to Doodle

 

Framadate

An easy-to-use libre Doodle-style event and poll service, provided by France’s largest ethical software organisation Framasoft. It costs nothing, it doesn’t track you and you don’t need to register.

If you enjoy using the service you can donate to Framasoft to help keep it going, but that’s entirely optional.
